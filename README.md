# Markdown 学习笔记



## 一、关于Markdown 

刚听说的时候我以为又是什么编译器，后面学习之后才知道是适用于很多地方的文字语言格式，很简单很牛逼，用几个符号就能加粗变大，用github离不开

## 二、必学基础语法

### 1. 标题

用`#`加空格开头，`#`越多标题越小（最多 6 级）



# 一级标题

## 二级标题

### 三级标题

and so on



### 2. 列表

#### 不知道叫啥，但可以用`-`/`*`/`+`加空格

瞎打打出来的



- 苹果
- 香蕉
  - 小香蕉
- 橘子
- 

### 3. 文字样式

这是在github看别人主页偷来的，还不懂原理，只是cv大神

```markdown
**加粗** → **我是粗体**
*斜体* → *我是斜体*
~~删除线~~ → ~~我是删除线~~
```

### 4. 链接

学这个是因为看到别人的README很简洁，可以直接跑去其他库

方法就是搞个库后依旧cv大神，搞到后面就行了

比如https://github.com/用户名/仓库/blob/分支/文件相对路径
